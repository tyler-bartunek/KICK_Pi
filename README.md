_Democratizing robotics, one shoebox at a time_

The KICK Robot framework is an open-source platform designed to make learning about and development of mobile robotic systems accessible, modular, and affordable.

## Overview
This repository contains specifically the code to be ran on the raspberry pi acting as the central controller for the system. You will need Docker installed on your Raspberry Pi, and it is recommended that you run the sh script first for building the container, as it will also set up all of the GPIO and get the necessary python packages installed as well as configuring ROS. 

For hardware files as well as the code to run on the peripheral pico controllers for each module, check out the [main repository](https://github.com/tyler-bartunek/KICK-Robot).
For the project documentation, please refer to the [wiki](https://github.com/tyler-bartunek/KICK-Robot/wiki)
